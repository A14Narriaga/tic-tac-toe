# Angular CLI
```
npm install -g @angular/cli
```
# Angular project instalation
```
npx ng new <project_name>
```
# Create component
```
npx ng g c <component_name>
```
# Add eslint format
```
npx ng add @angular-eslint/schematics
```
# Use format
```
npx ng lint --format
```
# Add PWA
```
npx ng add @angular/pwa 
```
